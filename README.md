# kyc-risk-scoring-engine
KYC Risk Rating 

**Install Streamlit:**
Run this command in your git bash 
git clone https://github.com/saranya90ramanathan-source/kyc-risk-scoring-engine
cd kyc-risk-scoring-engine
**Install Dependencies :**
python -m pip install streamlit
python -m pip install -r requirements.txt
**Run app**
python -m streamlit run app/app.py
